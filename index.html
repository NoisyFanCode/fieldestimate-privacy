FieldEstimate Privacy Policy

Last Updated: March 2026

This Privacy Policy explains how the FieldEstimate mobile application (“the App”) handles information. The App is developed and published by NoisyFanCode.

FieldEstimate is a construction material estimation calculator that allows users to perform calculations and save project estimates locally on their device.

Summary

  • No user accounts required
  • No personal data collection
  • Project data stored locally on the device
  • Anonymous crash diagnostics via Firebase Crashlytics
  • No advertising or tracking

1. Information Collection

The App is designed to minimize data collection.

Personal Information

FieldEstimate does not collect personal information such as:

  • Names
  • Email addresses
  • Phone numbers
  • Account information
  • Payment information
  • Project data

The App does not require user accounts.

Project and Calculation Data

Any data created inside the App, including:

  • Project folders
  • Calculation inputs
  • Generated estimates
  • Exported files (PDF or CSV)

is stored locally on the user's device only and is not transmitted to the developer or stored on external servers.

Users are responsible for managing and backing up their own exported files.

2. Crash Reporting and Diagnostics

To improve stability and performance, FieldEstimate uses Firebase Crashlytics, a crash reporting service provided by Google.

Firebase Crashlytics automatically collects anonymous diagnostic information when the app crashes or encounters errors.

This may include:

  • Device model
  • Operating system version
  • App version
  • Crash stack traces
  • Diagnostic information related to the crash

This data does not include:

  • Names
  • Email addresses
  • Project data
  • Calculation inputs
  • Personally identifiable information

Crash data is used solely to identify bugs and improve app reliability.

More information about Firebase privacy practices can be found at:

https://firebase.google.com/support/privacy

3. Use of Information

The limited diagnostic data collected via Firebase Crashlytics is used only to:

  • Identify application crashes
  • Diagnose technical issues
  • Improve application stability
  • Improve future versions of the App

The Developer does not sell, rent, or share user data.

4. Third-Party Services

The App uses the following third-party service:

Firebase Crashlytics
Provided by Google LLC

Purpose:

  • Crash reporting
  • Application diagnostics

Privacy information:
https://firebase.google.com/support/privacy

No other analytics, advertising, or tracking services are currently used.

5. Advertising and Tracking

FieldEstimate currently:

  • Does not display advertisements
  • Does not track users across apps or websites
  • Does not use advertising identifiers

6. Data Security

Because project data is stored locally on the user's device, the Developer does not have access to it.

Users are responsible for securing their own devices and managing exported files.

While reasonable efforts are made to maintain application stability and security, no method of electronic storage or transmission can be guaranteed to be completely secure.

7. Children’s Privacy

FieldEstimate is not specifically directed toward children under the age of 13.

The App does not knowingly collect personal information from children.

8. Changes to This Privacy Policy

This Privacy Policy may be updated from time to time.

When changes are made, the “Last Updated” date at the top of this policy will be revised.

Continued use of the App after updates constitutes acceptance of the updated policy.
